|----------|-------------|
| business logic to storage | Content/video/schedule code writes or reads object references. |
| file API to storage | Authenticated file serving resolves object references to bytes. |
| worker to storage | Worker process resolves local paths for FFmpeg/Python. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-04-04-02 | Information Disclosure | public URLs | mitigate | Keep file API ownership checks from Phase 3 and do not expose absolute paths. |
| T-04-04-03 | Denial of Service | storage deletes | mitigate | Preserve idempotent delete behavior and path assertions. |
| T-04-04-04 | Elevation of Privilege | worker path resolution | mitigate | Allow local path resolution only through storage service methods, not user-supplied paths. |

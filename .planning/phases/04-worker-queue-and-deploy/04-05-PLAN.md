<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| VPS env to containers | Secrets and service URLs enter runtime environment. |
| Docker volume to app/worker | Persistent media bytes are shared between web and worker. |
| public internet to app container | App is exposed through the VPS network/reverse proxy. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-05-01 | Information Disclosure | `.env.example` and docs | mitigate | Document names only; never include real secret values. |
| T-04-05-02 | Tampering | persistent media volume | mitigate | Mount the same storage root for app and worker and preserve storage path guards. |
| T-04-05-03 | Denial of Service | worker resource usage | mitigate | Run worker as separate service and document concurrency/runtime dependency checks. |
| T-04-05-04 | Repudiation | operations | mitigate | Document `docker compose logs worker app` and JobRun database diagnostics. |
</threat_model>

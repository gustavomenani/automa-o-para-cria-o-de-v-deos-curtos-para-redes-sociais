<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Redis to worker | Serialized job payload enters a privileged process. |
| worker to provider/process | Worker invokes external APIs, Python, FFmpeg, and social publisher boundary. |
| scheduler sweep to database | System job selects due posts without direct user interaction. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-02-01 | Spoofing | worker payload ownership | mitigate | Re-query project/schedule ownership before side effects. |
| T-04-02-02 | Tampering | schedule publish state | mitigate | Update ScheduledPost through deterministic handler transitions only. |
| T-04-02-03 | Repudiation | job execution | mitigate | Append JobRun logs for dispatch, major steps, completion, and failure. |
| T-04-02-04 | Information Disclosure | provider/process errors | mitigate | Persist normalized errors only; detailed console logs must not contain secrets. |
| T-04-02-05 | Denial of Service | worker concurrency | mitigate | Configure conservative worker concurrency and bounded retries in BullMQ options. |
</threat_model>

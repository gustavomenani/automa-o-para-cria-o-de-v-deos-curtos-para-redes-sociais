<objective>
Stabilize caption generation for the core MVP video path.

Purpose: Phase 1 is blocked by subtitle drift, incorrect Whisper text, awkward one-word cues, and unclear fallback quality. This plan makes caption behavior testable and preserves the in-progress local subtitle work instead of replacing it.

Output: Pure caption helpers, focused tests, and a VideoService integration that keeps FFmpeg in VideoService while using Whisper timestamps and project caption text correctly.
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| uploaded audio -> Whisper subprocess | Untrusted local media path is passed into Python/faster-whisper. |
| project caption/script -> ASS subtitle file | User/provider text is written into a subtitle file consumed by FFmpeg. |
| FFmpeg/ffprobe subprocess -> persisted status | External process failures influence `ContentProject.status`, `GeneratedVideo.status`, and user-facing warnings. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-01-01-01 | Tampering | `caption-helpers.ts` ASS escaping | mitigate | Escape braces, slashes, commas, ASS override syntax, and line breaks before writing cues; add tests proving malicious-looking caption text cannot inject ASS directives. |
| T-01-01-02 | Information Disclosure | `VideoService` error/warning persistence | mitigate | In both success-warning and catch-path persistence, write only stable Portuguese user-facing copy to `ContentProject.errorMessage`; never persist raw FFmpeg stderr, local paths, provider payloads, stack traces, or secret-bearing values. Keep raw details only in server-only logs if needed. |
| T-01-01-03 | Denial of Service | `transcription-service.ts` and `scripts/transcribe_audio.py` | accept for Phase 1 | Keep the existing transcription timeout and null fallback. Queue/worker isolation and upload duration limits are explicitly deferred outside Phase 1. |
| T-01-01-04 | Repudiation | Caption helper test coverage | mitigate | Add deterministic tests for helper outputs so future subtitle regressions can be traced to concrete behavior changes. |
</threat_model>
